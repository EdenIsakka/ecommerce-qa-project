# QA SUMMARY - E-COMMERCE PROJECT

## Descripcion general

Este poryecto evidencia el avance de un flujo completo de QA para un e-commerce real:
desde analisis funcioinales de prueba y error, hasta un reporte con sus defectos.
Se utilizo [SauceDemo] como entorno de practica.

##  Objetivos!!
- Aplicar buenas practicas de documentaciin QA manual  
- Identificar casos de prueba con prioridad y severidad adecuadas  
- Reportar defectos claros y reproducibles  
- Preparar la base para automatizacion con **Java + Selenium (TestNG)**
